## PM2, My Patronum, Let me reveal you

来，再来一遍(记得要中二破音）~~~ PM2，我的守护神，让我来揭秘你吧

PM2的原理，用一句话来概括，就是我们通过rpc跟守护进程通信,让守护进程帮我们去创建、销毁、重启进程，就这么简单。

但是，这中间，通信协议怎么去设计、进程怎么去维护等等这些，可没办法三眼两语讲完

所以，下面就跟着我的节奏一起摇摆吧~
